# rotten_tomatoes_mob
Une application mobile Flutter de découverte de films  construite avec une Clean Architecture, BLoC/Cubit  et l'API TMDB.   Fonctionnalités  -  Authentification (Login / Register / Logout) -  Liste des films populaires avec pagination infinie   Recherche de films en temps réel -  Filtrage par catégorie / genre - Gestion des favoris (stockage local)
